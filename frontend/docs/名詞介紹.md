# 名詞介紹

> 本專案同時存在「Router 驅動的傳統頁」與「Tab 渲染型元件」兩種畫面，
> 為避免溝通時名詞打架，統一以下用詞。核心規則：**「頁面」這個模糊詞退休，一律改用專屬名稱。**

## 名詞對照表

| 名詞 | 定義 | 對應位置 / 備註 |
|---|---|---|
| 首頁 | 使用者登入後第一個顯示的畫面 | React 頁面對應 `src/routes/_layout.tsx` 與其首頁子路由 `src/routes/_layout/index.tsx` |
| **路由頁** | 會改 Router / 網址的傳統頁，Router 驅動，切換它網址會變 | 走 React Router |
| **頁籤** / **Tab** | 放在 Tab 裡渲染的元件，狀態驅動，不碰 Router | 未來大量使用 |
| **頁籤標籤** / tab chip | 頁籤列上那個可點、可關的小東西 | 視覺上那一條 |
