# 專案介紹

本專案 `my-full-stack-fastapi/frontend` 是基於 [fastapi/full-stack-fastapi-template](https://github.com/fastapi/full-stack-fastapi-template) 的前端，目前維持官方模板的原始結構（尚未加入自訂頁面或元件）。

## 技術棧

| 類別 | 技術 |
|---|---|
| 框架 | React 19 + Vite 7 + TypeScript |
| 路由 | TanStack Router（檔案式路由，位於 `src/routes/`） |
| 資料層 | TanStack Query（react-query） |
| UI | Tailwind CSS 4 + Radix UI + shadcn 風格元件（`src/components/ui/`） |
| 表單 | react-hook-form + zod |
| API Client | 由 `openapi-ts` 自動生成（`src/client/`），對應後端 OpenAPI schema |
| 測試 | Playwright E2E |
| Lint | Biome |
| 套件管理 | Bun |

## 目錄結構

```text
src/
├── client/        # 自動生成的 API client（OpenAPI）
├── components/
│   ├── Admin/     # 後台使用者管理（新增/編輯/刪除）
│   ├── Items/     # Items CRUD
│   ├── Pending/   # 載入中的 skeleton 元件
│   ├── Sidebar/   # App 側邊欄
│   ├── UserSettings/ # 個人設定（改密碼、刪除帳號）
│   ├── Common/    # 共用元件（Logo、Footer、AuthLayout 等）
│   └── ui/        # shadcn 基礎元件
├── hooks/         # useAuth、useCustomToast、useMobile 等自訂 hooks
└── routes/        # 頁面路由
```

## 路由一覽

| 路徑 | 說明 |
|---|---|
| `/login`、`/signup` | 登入 / 註冊 |
| `/recover-password`、`/reset-password` | 忘記密碼流程 |
| `/_layout` | 登入後主版面（含側邊欄） |
| `/_layout/` | 首頁 / Dashboard |
| `/_layout/items` | Items 管理 |
| `/_layout/admin` | 使用者管理（Admin） |
| `/_layout/settings` | 個人設定 |

## 開發指令

```bash
bun install
bun run dev      # 啟動開發伺服器（http://localhost:5173）
bun run build    # TypeScript 編譯 + Vite build
bun run lint     # Biome 檢查並自動修正
bun run generate-client  # 重新產生 API client（需先取得後端 openapi.json）
bun run test     # Playwright E2E 測試
```
