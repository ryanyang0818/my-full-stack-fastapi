# 首頁公告橫幅執行計畫

## 計畫內容

- 在 `src/routes/_layout/index.tsx` 首頁最上方新增固定公告橫幅。
- 公告標題：`歡迎使用管理平台`
- 公告內文：`你可以從左側選單開始管理 Items、帳號與個人設定。`
- 使用淡 `primary` 背景、品牌色邊框、圓角及 Lucide icon。
- 支援響應式版面及亮色／暗色主題。
- 保留原有使用者歡迎訊息，排列於公告下方。
- 將登入後共用 Header 改為橫跨 Sidebar 與內容區的固定頂欄。
- 桌面版 Sidebar 從 Header 下方開始，行動版維持抽屜式 Sidebar。
- 執行 `bun run build`，並檢查首頁、Sidebar、路由及登入機制。

## 補充回答

否。沿用專案現有的 `lucide-react`、Tailwind CSS 與 React。
