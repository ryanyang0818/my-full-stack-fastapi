# Dodo Admin 常見問題

## Dodo Admin 是 ERP 嗎？

不是。

Dodo Admin 是企業後台管理系統的基礎平台，
重點是提供登入、權限、選單、版面、
資料管理與資料表等共用能力。

它可以支撐未來的業務模組，
但本身不等於完整 ERP。

## Dodo Admin 目前的核心範圍是什麼？

Dodo Admin 的核心範圍，
是建立所有後台管理功能都會用到的基礎能力。

包含：

- 登入
- 權限
- 選單
- 後台版面
- 資料管理
- 資料表
- 共用操作流程

## 使用者登入後，是不是所有功能都能使用？

不是。

登入只代表系統確認使用者身份。

使用者能看到哪些功能，
還要依照角色、選單權限與個人例外決定。

## 前端隱藏選單，就代表安全了嗎？

不是。

前端隱藏選單是一種使用體驗控制，
真正的資料操作仍需要由後端確認權限。

## 可以加入採購、訂單、庫存這類功能嗎？

可以。

這些可以作為可擴充的業務模組加入。

但它們應該建立在 Dodo Admin 的基礎能力之上，
而不是一開始就把系統定義成完整 ERP。

## 為什麼文件裡會提到 AI agent？

因為 Dodo Admin 的目標之一，
是讓工程師與 AI agent 都能穩定理解並接手專案。

清楚的文件、固定的資料夾結構與一致的開發規則，
能降低交接成本，也能讓 AI 協作更穩定。
