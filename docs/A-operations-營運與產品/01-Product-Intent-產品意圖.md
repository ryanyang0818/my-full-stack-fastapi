# Dodo Admin 產品意圖

Dodo Admin 是一套給企業內部管理系統使用的 full-stack admin foundation。

它的目的不是一次做成完整 ERP，而是先建立一個穩定、清楚、可擴充的後台系統骨架，讓後續功能可以用一致的方式被加入。

## 核心意圖

- 提供登入、權限、選單、版面、CRUD、資料表等後台基礎能力
- 讓工程師可以快速建立新的管理模組
- 讓管理者可以清楚控制使用者能看到與操作的功能
- 讓 AI agent 能透過文件、規則與一致架構穩定協作

## 邊界

Dodo Admin 不是完整 ERP。

它目前不追求複雜的會計、庫存稽核、製造流程或大型企業流程編排，而是優先把 admin 系統的共同基礎打穩。
